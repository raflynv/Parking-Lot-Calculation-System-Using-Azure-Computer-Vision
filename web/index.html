Parking Space Counter with Azure Custom Vision

Parking Space Counter ini merupakan implementasi dari Computer Vision dimana menggunakan Object Detection. Project ini diperuntukan untuk petugas parkir yang kesulitan dalam menemukan lahan kosong dimana nantinya dapat mempercepat mobilisasi dari antrian mobil. Parking Space Counter ini menggunakan layanan Azure Custom Vision, Azure ML dan Visual Studio Code

Solusi ini mungkin masih banyak kekurangan karena memang ada keterbatasan baik tenaga maupun waktu, namun ini bisa menjadi salah satu langkah agar
bisa digunakan di kasus realnya. Terima kasih
